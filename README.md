# Cavane website

Static GitHub Pages site for `cavane.mx`.

## Files to upload
Upload everything in this folder to the root of the GitHub repository:

- `index.html`
- `404.html`
- `CNAME`
- `.nojekyll`
- `robots.txt`
- `sitemap.xml`
- `llms.txt`
- `cavane-facts.json`
- `site.webmanifest`
- `/assets`

Do not upload only `index.html`. That is how websites become modern cave paintings.

## Custom domain
The `CNAME` file already contains:

```txt
cavane.mx
```

If you decide to use `www.cavane.mx` as canonical instead, change the `CNAME`, canonical URLs, sitemap, robots and structured data.
